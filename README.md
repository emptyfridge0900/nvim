### "
last yank/delete
### "0p 
Paste from register 0
### "aY
Yanks a line into the a register
### ciw 
cut inner word
### ci" 
cut inner ""
### ci< 
cut inner <>
### di" 
delete inner ""
### da" 
delete "" and inner ""
### ysiw"
### ysi("
### cs[( 
change surrounding [ to (
### * 
goes to the next matching word
### \# 
goes to the previous matching word
### ;
reapeat the command
### f{char} F{char}
다음 나타나는 char로 가기
> f(;  
> cfo  
> df(  
> d2f(  
### t{char} T{char}
다음 나타나는 char 앞으로 가기
> cto
> dt(
